# Slide Translator Skill

分子生物学の研究発表スライド（.pptx）内の日本語テキストを抽出し、適切な英語表現に翻訳して置換するスキルです。一般的な分子生物学の研究発表に特化した翻訳ガイドラインを備えています。

## 特徴

- **日本語テキストの自動抽出**: スライドを解析し、翻訳が必要な箇所だけを抽出します。
- **コンテキストベースの翻訳**: 抽出時には前後の英語テキストも表示され、文脈に応じた正確な翻訳をサポートします。
- **フォントの自動最適化**: 翻訳された箇所のフォントを強制的に「Arial」へ変更し、レイアウト崩れやフォントの不整合を防ぎます。
- **効率的な実行**: トークン消費を抑えるため、抽出と適用の最小限のステップ（2コマンド）で完結するように設計されています。

## 前提条件

このスキルは裏側で`pptxスキル` (https://github.com/anthropics/skills/tree/main/skills/pptx)を利用してファイルの展開・再パッケージ化を行っています。
利用する環境に`pptx`スキルがインストールされていることを確認してください。

## 使い方

エージェントに「スライドを英訳して」「プレゼンを翻訳して」と依頼するだけで自動的にワークフローが開始されますが、内部的には以下のステップで処理されます。

### 1. 日本語テキストの抽出

```bash
python scripts/extract_japanese.py <input.pptx> <work_dir> [--exclude-slides 1 2 ...]
```

- 指定したPPTXファイルを展開し、日本語が含まれるテキストボックスを抽出します。
- 抽出結果は `<work_dir>/japanese_texts.json` に保存されます。
- `--exclude-slides` オプションで、タイトルスライド（例：1ページ目）など翻訳不要なスライドを除外できます。

### 2. 翻訳ファイルの作成 (エージェントが自動実行)

抽出されたJSONをもとに、エージェントが `translations.json` を作成します。この際、以下のガイドラインに従って翻訳が行われます。

- 遺伝子やタンパク質の一般名称（例: Gene A, Protein B）はそのまま英語として扱います。
- 一般的な分子生物学の略称（WT, GFPなど）を適切に使用します。

### 3. 翻訳の適用とファイルの再構築

```bash
python scripts/apply_translations.py <work_dir> translations.json <output.pptx> --original <input.pptx>
```

- 日本語テキストを英語に置換します。
- 翻訳箇所の言語設定を英語（en-US）に変更します。
- 翻訳箇所のフォントをすべて「Arial」に変更します。
- 展開されたファイルを再び `.pptx` 形式にパッケージ化します。

### 4. 確認とクリーンアップ

生成された出力ファイルを確認し、問題がなければエージェントが一時ディレクトリ（`<work_dir>`）を削除（クリーンアップ）して作業を完了します。

## 対象外となるもの

- 発表者ツール用の「ノート」テキストは翻訳対象外（現状の仕様）です。
- 元から英語のみで記述されているテキストボックスのフォントや内容は変更されません。
